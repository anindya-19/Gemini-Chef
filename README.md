# 🍳 Chef Gemini

A React web application that uses Google's **Gemini AI** to suggest recipes based on ingredients you have on hand. Simply type in what's in your fridge and pantry, and let Chef Gemini do the rest!

## ✨ Features

- **Ingredient Input** – Add ingredients one-by-one via a simple input form
- **Live Ingredient List** – See all your added ingredients displayed in real time
- **AI-Powered Recipe Suggestions** – Once you've added at least 4 ingredients, request a recipe generated by Gemini AI
- **Markdown Rendering** – Recipes are formatted beautifully using `react-markdown`

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 19, JSX |
| Build Tool | Vite |
| AI Integration | Google Gemini (`gemini-3.1-flash-lite-preview`) |
| Styling | Vanilla CSS (Inter font via Google Fonts) |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- A [Google Gemini API key](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd "Gemini Chef"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**

   Create a `.env` file in the project root:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

   > ⚠️ **Important:** Never commit your `.env` file. It is already listed in `.gitignore`.

4. **Start the development server:**
   ```bash
   npm run dev
   ```

   Open your browser and navigate to `http://localhost:5173`.

## 🧠 How It Works

1. The user types ingredients into the input form and clicks **Add ingredient**.
2. Once at least **4 ingredients** are added, a **"Get a recipe"** button appears.
3. Clicking the button calls `getRecipeFromGemini()` in `ai.js`, which sends the ingredient list to the Gemini API with a system prompt instructing it to suggest a fitting recipe.
4. The AI response (formatted in Markdown) is rendered on screen via the `GeminiRecipe` component.
